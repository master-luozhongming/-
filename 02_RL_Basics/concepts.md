# 强化学习基本概念

## 核心要素

### 智能体与环境

```
┌─────────────┐         ┌─────────────┐
│   智能体     │ ──────► │    环境      │
│   (Agent)    │ 动作a   │(Environment)│
│              │ ◄────── │             │
└─────────────┘ 状态s,r  └─────────────┘
```

- **智能体 (Agent)**: 学习和决策的主体
- **环境 (Environment)**: 智能体交互的对象

### 状态、动作、奖励

| 概念 | 符号 | 含义 |
|------|------|------|
| 状态 | s, S_t | 环境的当前状态 |
| 动作 | a, A_t | 智能体采取的行为 |
| 奖励 | r, R_t | 即时反馈信号 |

---

## 策略 (Policy)

### 定义
策略是状态到动作的映射函数：

```
π(a|s) = P(a_t = a | s_t = s)
```

### 策略的类型

1. **确定性策略**: a = π(s)
2. **随机策略**: a ~ π(·|s)

### 示例：倒立摆

```python
# 策略函数输出动作的概率分布
π(向左推|s) = 0.7
π(向右推|s) = 0.3

# 从分布中采样
action = sample_from_distribution(π(·|s))
```

### 深度强化学习
当策略函数是神经网络时：
```
深度学习 + 强化学习 = 深度强化学习
```

---

## 轨迹 (Trajectory)

### 定义
一条轨迹是状态、动作、奖励的序列：

```
τ = (S_0, A_0, R_0, S_1, A_1, R_1, S_2, A_2, R_2, ...)
```

### 示例
```
τ = (向左推, 向右推, 向右推, 游戏失败)
```

由于动作是从概率分布中采样的，所以会有很多条不同的轨迹。

---

## 状态转移概率

### 马尔可夫性质
```
p(s'|s, a) = p(s_t | s_{t-1}, a_{t-1})
```

下一个状态只取决于当前状态和动作，与历史状态无关。

### 状态转移是智能体无法控制的
环境的状态转移是随机的，智能体无法控制。

---

## 即时奖励

### 奖励函数
```
r(s, a, s') 或 R_t = r(s_t = s, a_t = a, s_{t+1} = s')
```

### 奖励设计的挑战
- **奖励黑客 (Reward hacking)**: 奖励太容易获得
- **奖励稀疏性**: 奖励太难获得

---

## 回报 (Return)

### 定义
从t时刻起，未来获得的累积折扣奖励：

```
G_t = R_t + γR_{t+1} + γ²R_{t+2} + ...
    = R_t + γG_{t+1}
```

### 折扣因子 γ
- γ ∈ [0, 1]
- 使近期奖励更重要
- 防止回报发散到无穷大

### 示例
```
γ = 0.9
G_t = R_t + 0.9R_{t+1} + 0.81R_{t+2} + ...
```

---

## 价值函数 (Value Function)

### 状态价值函数 V(s)
在状态s下，按照策略π行动，未来预期回报：

```
V_π(s) = E_π[G_t | S_t = s]
       = E_π[∑_{k=0}^∞ γ^k R_{t+k} | S_t = s]
```

### 动作价值函数 Q(s,a)
在状态s下采取动作a，然后按照策略π行动，未来预期回报：

```
Q_π(s, a) = E_π[G_t | S_t = s, A_t = a]
```

### 关系
```
V_π(s) = ∑_a π(a|s) Q_π(s, a)
```

---

## 贝尔曼方程

### 贝尔曼期望方程
```
V_π(S_t) = E_π[R_t + γV_π(S_{t+1}) | S_t]
```

### 直观解释
```
当前状态的价值 = 即时奖励 + 折扣因子 × 下一状态的价值
```

### 推导
```
V_π(s) = E_π[G_t | S_t = s]
       = E_π[R_t + γG_{t+1} | S_t = s]
       = E_π[R_t + γV_π(S_{t+1}) | S_t = s]
```

## 探索与利用

### 探索 (Exploration)
尝试新的动作，可能发现更好的策略

### 利用 (Exploitation)
选择当前已知最好的动作

### 平衡
- 纯利用：可能陷入局部最优
- 纯探索：无法获得稳定收益
- **ε-贪心策略**: 以ε概率随机探索，1-ε概率利用

```python
def epsilon_greedy(q_values, epsilon=0.1):
    if random.random() < epsilon:
        return random.randint(0, len(q_values) - 1)  # 探索
    else:
        return np.argmax(q_values)  # 利用
```

## 倒立摆环境

### 环境描述
- **状态**: [推车位置, 推车速度, 木杆角度, 木杆角速度]
- **动作**: 向左推(0), 向右推(1)
- **奖励**: 每步+1（保持平衡）
- **结束条件**: 杆子倒下或超出范围

### 代码示例

```python
import gym

# 创建环境
env = gym.make("CartPole-v0")

# 重置环境
state = env.reset()
print(f"初始状态: {state}")  # [位置, 速度, 角度, 角速度]

# 执行一步
action = 1  # 向右推
next_state, reward, done, info = env.step(action)
print(f"下一状态: {next_state}")
print(f"奖励: {reward}")
print(f"是否结束: {done}")
```

## 关键要点

1. **核心循环**: 智能体与环境交互：状态 → 动作 → 奖励 → 新状态
2. **策略**: 决定在什么状态下采取什么动作
3. **价值函数**: 评估状态或状态-动作对的好坏
4. **贝尔曼方程**: 动态规划的基础
5. **探索与利用**: 平衡当前收益和长期收益
